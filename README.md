PiProbe is a thin linux distribution for use as a headless network probe based on the Raspberry PI hardware and the MINIBIAN build. 
https://minibianpi.wordpress.com/

- IPtables
- SSHGuard
- SSHd
- nmap
- mtr
- dnsutils
- dnstracer
- IPv6
- VLAN support
- etckeeper
